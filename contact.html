FAQ

Frequently Asked Questions

How does the AI match score work? ▼
The AI engine uses a 4-factor scoring model: technical skills overlap (40%), preferred domain match (25%), CGPA vs minimum requirement (20%), and areas of interest alignment (15%). These combine for a score out of 100%.
How do I login as Admin? ▼
Go to the Login page and use these credentials — Email: admin@aiias.edu / Password: Admin@1234. After login you will be redirected to the Admin Dashboard automatically.
How do I register as a Student? ▼
Click Login on the navbar, then switch to the Register tab. Fill in your name, email and password. After registration you'll be redirected to the Student Dashboard. Then fill your profile in Student Management to get AI recommendations.
Where is student data stored? ▼
All data is saved in your browser's LocalStorage — this means it works 100% offline without any backend server. The data persists across page reloads but is browser-specific. The Node.js/MongoDB backend can be connected for cross-device persistence.
Can I use this on GitHub Pages? ▼
Yes! The entire frontend works on GitHub Pages using LocalStorage. No backend is required for the frontend to function. If you want MongoDB storage across devices, deploy the backend folder to Render.com (free tier) and update the API_BASE in api.js.
Is this suitable for a viva presentation? ▼
Absolutely. The system demonstrates AI/ML concepts, role-based authentication, full-stack web development, database design, REST API architecture and modern UI/UX — all core topics examiners look for in a final year capstone project.
